Usando audios

abaixo é possível ver a tag <audio> em ação
para que o audio toque sozinho se usa o parametro "autoplay", e para aparecer o menu de controle de audio se usa "controls"
Essa tag suporta os formatos de audio "MP3, WAV, OGG"

agora aqui a gente vai ver a forma mais complexa de fazer esse lance de usar audios(entra no código pra ver melhor)
Dá pra usar o parametro "loop" dentro da tag <audio> pra reiniciar o audio, se tiver mais de um <source> dentro da tag <audio> ele vai tentar carregar na ordem disposta dentro da tag(não quando terminar um audio ele n vai começar o outro... eu acho)
