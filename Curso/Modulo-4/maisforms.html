Usando umas coisas de form que não são inputs

Começando por Select, datalist e textarea

A tag select

Essa tag é aquelas listas que você pode escolher uma opção, tipo aquelas de escolher estado, aí pra gente dar as opções da lista, dentro do select a gente pões uma tag OPTION pra cada opção, e outro detalhe legal é que podemos usar a tag optGroup pra agrupar opções em uma subLista, pra darmos o nome dessa subLista a gete coloca um label dentro da tag OPTGROUP

A tag datalist

Pra usarmos o datalist primeiro colocamos um input de texto e nele nos colocamos a propriedade List nessa propriedade deverá ser indicado o ID da lista que vai ser usada no campo, após isso usamos a tag datalist, e dentro dela colocamos as option, essa tag é util para quando queremos dar opçoes pro usuario, mas tambem dar a opção de ele informar umaopção que pode não estar presente

Usando o TextArea

É recomendado usar o textarea em forms com o metodo post, o textarea cria um campo para o usuario digitar um texto

Falando de Outputs

Nos outputs nos colocamos valores e etc que serão gerados a partir de respostas nos inputs dos forms, pra isso nós vamos no input que vai dar valor e etc ao output e colocamos a propriedade oninput="codigo em JS". Coisa importante de output eles não servem pra enviar dados apenas pra mostrar
